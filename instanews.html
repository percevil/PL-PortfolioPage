INSTANEWS

A New York Times content collection app

  • see it live
  • see the code

Instanews is a content collecting app for the New York Times newspaper website. It collects content
dynamically from various interests through their API. The app was written with Javascipt, jQuery and
some AJAX to search and display the articles. Animations were accomplished with CSS and adds some
flair to an otherwise plain home screen.
