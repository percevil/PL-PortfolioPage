Smart RCIC

A mobile app that helps students become immigration consultants

  • see a demo
  • read the case study

Smart RCIC was my first client project at Red. The clients were software developers that
were making an app for students studying to become immigration consultants.
They wanted us to design them a purpose built user interface that had the look and image
that they were looking for. This project was another live agency scenario using the Agile
software development methodologies. Our sprint cycle was 3 weeks, with scrums every morning.
