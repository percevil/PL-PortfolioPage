Simple Styles Apparel CO

A website landing page re-designed and enhanced from a previous project

  • see it live
  • see the code

The Simple Styles Apparel Co. website is a re-designed web site landing page from one of my first
projects at Red Academy.Its uses HTML, CSS and CSS variables, Sass and my first experiences
with using linters. Vanilla Javascipt was used for the form, but I added more using Greensock's
Gsap animation library to animate the new company logo. The website was written to be fully responsive.
