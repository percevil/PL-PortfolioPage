DOTA Heroes filter app

A character filtering app based on the Action RTS video game DOTA2 Heroes

  • see it live
  • see the code

DOTA2 Heroes filtering app is a project I wrote after graduating from Red Academy in efforts to
further strengthen my understanding of vanilla Javascript. This app was one of the projects from an
online course I enrolled in. It uses Asynchronous (AJAX) Javascript principles to filter information on
particular characters based on three categories. This information is collected from Dota2's api. The UI
was pre-written by the program providers. In future I would like to re-design the interface to my specifications.
